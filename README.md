
#  Hospital Management System

##  Overview

This is a desktop-based Hospital Management System developed using Java and Swing. It helps manage hospital operations such as patient records, doctor details, appointments, and pharmacy management.

---

##  Features

* 👨‍⚕️ Doctor Management (Add, View, Update)
* 🧾 Patient Records Management
* 🏥 Appointment Scheduling
* 💊 Pharmacy Module
* 🔐 Login System for different users (Doctor, Receptionist, Pharmacist)

---

##  Technologies Used

* Java (JDK 8 or above)
* Swing (GUI)
* MySQL (Database)
* JDBC (Database Connectivity)

---

##  Project Structure

* `src/` → Java source files
* `lib/` → External JAR files 
* `database.sql` → Database file

---

## ⚙️ Requirements

* Java JDK 8+
* Eclipse IDE (or any Java IDE)
* MySQL Server

---

## 🧑‍💻 How to Run the Project

### Step 1: Clone Repository

```
https://github.com/rahulbaliarsingh/HospitalManagementSystem
```

### Step 2: Import Project

* Open Eclipse
* File → Import → Existing Projects into Workspace

### Step 3: Add Libraries

* Right-click project → Build Path → Add External JARs
* Add all JAR files from `lib/` folder

### Step 4: Setup Database

* Open MySQL
* Create database:

```
CREATE DATABASE hospital;
```

* Import `database.sql` file

### Step 5: Run Project

* Open main class (e.g., `Login.java`)
* Run as → Java Application
